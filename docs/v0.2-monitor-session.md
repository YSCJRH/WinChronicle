# v0.2 Monitor Session

The v0.2 monitor session is an explicit, finite, local-only path for turning
watcher events into a human-readable activity timeline and deterministic
suggestions.

It is not a daemon, service, startup task, polling loop, screen recorder, or
desktop control tool.

## Commands

Deterministic fixture session:

```powershell
python -m winchronicle monitor --events harness/fixtures/watcher/notepad_burst.jsonl --session-id demo
python -m winchronicle summarize-session demo
```

Explicit watcher session:

```powershell
python -m winchronicle monitor --watcher dotnet --watcher-arg resources/win-uia-watcher/bin/Debug/net8.0-windows/win-uia-watcher.dll --helper dotnet --helper-arg resources/win-uia-helper/bin/Debug/net8.0-windows/win-uia-helper.dll --duration 60 --capture-on-start
```

## Output

- Session JSON is written under the local state `sessions` directory.
- A local HTML report is written under the local state `reports` directory.
- `summarize-session` reads a saved session id from the local state directory;
  it is not an arbitrary JSON file reader.
- Captures still go through the existing normalize, privacy, redaction, schema,
  storage, and SQLite indexing path.
- Raw watcher JSONL is not saved by the product path.

## Deterministic Suggestions

The first v0.2 suggestions are deterministic and intentionally simple:

- repeated unchanged UI state,
- frequent multi-app context switching,
- error-like observed text,
- privacy or operator skipped windows.

These are local heuristics, not LLM summaries.

## Operator Controls

Use `--exclude-app <name>` to skip exact app names for a single monitor
session. Excluded windows are counted as skipped, and their observed content is
not persisted by the monitor session.

## Boundaries

The monitor session does not add screenshots, OCR, audio recording, keylogging,
clipboard capture, cloud upload, desktop control, MCP write tools,
daemon/service installation, default background capture, or product targeted
capture by HWND, PID, title, or process name.
